# Repository: "Linking human brain functional connectivity to underlying neurotransmission"

This repository will contain the code and sharable data for our introduction and validation study of a new method to study associations between brain connectivity and neurobiological atlases (e.g. PET or mRNA expression maps).  
Here, we refer to this framework as *Neurobiological Organization of Functional Connectivity: **NEOFC***.

This repository contains code and all openly sharable data.
You can find a conference poster [here](https://leondlotter.de/doc/poster/2025_WFSBP_PETfc.pdf).
