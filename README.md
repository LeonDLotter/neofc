# Repository: "Molecular predictors of human brain connectome organization (working title)"

This repository will contain the code and sharable data for our introduction and validation study of a new method to study associations between brain connectivity and neurobiological markers (e.g. PET or mRNA expression maps).

This repository will contain code and parts of the data as soon as the preprint is out (expected by the end of 2026).
You can find a recent conference poster [here](https://leondlotter.de/doc/poster/2025_WFSBP_PETfc.pdf).
